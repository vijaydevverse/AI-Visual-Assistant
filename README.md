# 🧠 AI Visual Assistant (YOLOv8 + Llama 3)

A real-time AI-powered vision system that detects objects using **YOLOv8** and explains them using **Llama 3 (Ollama)** with voice output.

---

## 🚀 Features

- 📸 Press **S** to capture image
- 🧠 Detects objects using YOLOv8
- 🤖 Explains objects using Llama 3 (Ollama)
- 🔊 Voice output using pyttsx3
- 🎯 Snapshot-based AI analysis (no continuous spam detection)
- 🖥️ Real-time camera preview

---

## 🛠️ Tech Stack

- Python
- OpenCV
- Ultralytics YOLOv8
- Ollama (Llama 3)
- pyttsx3
